## TONE RULES FOR FUTURE COPY

1. Never use "Wir glauben" ‚Äî show beliefs through tone
2. Never use "unglaublich lecker" ‚Äî everyone says this  
3. Never use "Zukunft" ‚Äî overused in plant-based
4. Write like you'd talk to a friend
5. One self-deprecating moment per major section
6. Acknowledge the weird when it's weird
7. Data is good; smug data is bad
8. If you wouldn't say it at a BBQ, don't write it
